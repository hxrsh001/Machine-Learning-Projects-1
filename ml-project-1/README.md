# ML Project 1 - Regression Algorithms

## 📌 Overview
This repository contains multiple Machine Learning regression models implemented using Python and Scikit-learn. Each notebook demonstrates a different regression algorithm on real-world datasets.

## 📂 Project Structure

```
ML-PROJECT-1
│
├── LinearReg.ipynb
├── MultiLinReg.ipynb
├── PolynomialReg.ipynb
├── RidgeReg.ipynb
├── LassoReg.ipynb
├── ElasticNetReg.ipynb
│
├── employee_salary_data.csv
├── house_data.csv
├── student_data.csv
├── student_performance.csv
│
└── README.md
```

## 🚀 How to Run

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install the required packages

```bash
pip install -r requirements.txt
```

3. Open any `.ipynb` notebook in VS Code or Jupyter Notebook.

4. Run the notebook cells from top to bottom.
